HG Changelog Parser

Kilka informacji:

Autor: Siof
Repozytorium: https://bitbucket.org/siof/hg-changelog-parser
Typ: Aplikacja konsolowa
Język: C++
System operacyjny: GNU/Linux, windows


Opis:

Mała aplikacja która powstała na potrzeby uaktualnienia oraz poprawnienia logu zmian projektu HellGround.

Parsuje ona repozytorium mercuriala oraz pozwala na wygenerowanie pliku historii posiadajacego:

  • pełną listę poprawek
  • historię od konkretnego numeru poprawki
  • konkretną ilość ostatnich poprawek

Lista zmian jest zapisywana w formacie określonym przez użytkownika, jednak musi on zawierać 3 następujące po sobie dane:

  • Numer reva (poprawki) - unsigned int - %u
  • Autor - string - %s
  • Opis - string - %s

Pomiędzy danymi może być jakikolwiek inny tekst nie będący tagami formatowania tekstu w C.
Domyślnym formatem jest: [%u %s] %s
