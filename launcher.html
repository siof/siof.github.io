HellGround Launcher

Kilka informacji:

Autorzy: lukaasm, Siof
Repozytorium: http://dev.hellground.pl/launcher
Typ: Aplikacja desktopowa
Język: C++
System operacyjny: GNU/Linux, Windows


Opis:

Launcher przygotowany dla projektu HellGround. Aplikacja wyświetla podaną stronę www jako swoją zawartość oraz dostosowywuje rozmiar okna to wielkości tej strony. W przypadku niemożliwości załadowania strony zostanie wyświetlona standardowa grafika. Aplikacja działa pod systemami z rodziny GNU/Linux oraz windows wykorzystując bibliotekę QT do stworzenia UI.
